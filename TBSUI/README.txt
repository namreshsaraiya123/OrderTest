Open Visual Studio
Open the TBSUI.sln file from the TBSUI folder
Do Build -> Clean Solution
Do Build -> Solution

Open command prompt as admin
Navigate to your TBSUI folder
RunOrderTest.cmd

This will run all the test given in the challenge.
The results will be seen in the cmd and also html reports will be generated in Test Results Folder




Note: Make sure .Net Framework is 4.5.2
All the nuget package such as selenium webdriver are installed properly.
